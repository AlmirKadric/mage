Interface IArchivistTopicVaultConfiguration

Define how data will be read, serialized and deserialized.

Other configuration attributes may be available depending on the vault backend being used by the parent topic.

Please see https://mage.github.io/mage/api.html#vaults, and look for the paragraph called "Required topic API" for your current vault; any functions defined there can be overridden for your given topic.
